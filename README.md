# Assignment 3 – Personal Portfolio (Advanced Features)

## Project Description
This project is an enhanced version of my personal portfolio website from previous assignments. In this version, I added more advanced interactivity, including API integration, improved validation, and better user experience using JavaScript and dynamic content.

## Features
- About, Projects, and Contact sections  
- Dark/light theme toggle with saved preference using localStorage  
- Live project search feature  
- Contact form validation with user feedback  
- Fetch and display data from GitHub API (latest repositories)  
- Error handling for API requests  
- Responsive design for desktop, tablet, and mobile  
- Hover effects and smooth transitions for better user experience  

## Folder Structure
- `index.html` – main page  
- `css/styles.css` – styling, layout, responsiveness, and animations  
- `js/script.js` – theme toggle, search, form validation, and API feature  
- `assets/images/` – placeholder images  
- `docs/ai-usage-report.md` – AI usage report  
- `docs/technical-documentation.md` – technical details  

## How to Run Locally
1. Clone the repository:
   - `git clone https://github.com/dalivxs/202282360-DaliaAlquwaidhi-assignment3`
2. Open the folder in VS Code.
3. Run using Live Server OR open:
   - `http://127.0.0.1:5500/index.html`
