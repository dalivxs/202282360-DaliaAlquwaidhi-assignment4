# Assignment 4 – Personal Portfolio (Final Application)

## Project Description  
This project is the final version of my personal portfolio web application. It combines all features developed in previous assignments into a complete, polished, and interactive website. The application demonstrates front-end development skills, API integration, state management, and advanced JavaScript logic while providing a professional user experience.

## Live Demo  
https://dalivxs.github.io/202282360-DaliaAlquwaidhi-assignment4/

## Features  
- About, Projects, and Contact sections  
- Dark/light theme toggle with saved preference using localStorage  
- Visitor name storage and personalized message  
- Live project search feature  
- Project sorting (name and year)  
- Project filtering by level (Beginner / Advanced) with dynamic messages  
- Contact form validation with multiple checks and user feedback  
- Timer showing how long the user has been on the website  
- Fetch and display GitHub repositories using GitHub API  
- Error handling for API requests  
- Responsive design for desktop, tablet, and mobile  
- Smooth transitions and modern UI design  

## Folder Structure  
- `index.html` – main page  
- `css/styles.css` – styling, layout, responsiveness, and animations  
- `js/script.js` – logic (theme, search, sorting, filtering, API, state management)  
- `assets/images/` – images and placeholders  
- `docs/ai-usage-report.md` – AI usage report  
- `docs/technical-documentation.md` – technical details  
- `presentation/` – presentation slides and demo video  
- `.gitignore` – ignored files configuration  

## How to Run Locally  
1. Clone the repository:  
   `git clone https://github.com/dalivxs/202282360-DaliaAlquwaidhi-assignment4`
